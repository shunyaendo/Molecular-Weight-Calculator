# kadai_SE

分子式を入力すると、分子量を計算する。  

## デモ

例：H2Oと入力し、計算ボタンをクリックすると、分子量：18.016と計算される。

![JavaScript-Practice](https://user-images.githubusercontent.com/110656746/212015437-acf2b346-47f9-4dbf-a173-b31508a3eaa5.gif)

## 使い方

1. 分子式を入力する。
2. 計算ボタンをクリックする。
3. 分子量が表示される。

## 注意事項

- 分子式は半角で入力すること。
- 有効数字が統一されていないこと（入力した内容により、有効数字が異なる）。
